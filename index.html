Tentang

Hallo! Perkenalkan saya Fani Nur Azizah, orang-orang biasa memanggil saya Fani, saya lahir di Tegal 31 Oktober 2003, saya berdomisili Bojong, saya anak terakhir dari 2 bersaudara.

Saat usia 7 tahun, saya bersekolah di SDN Tuwel 03. Setelah lulus SD, saya diterima di SMPN 01 Bojong. Kemudian, saya melanjutkan sekolah di SMA N 1 BOJONG. Saat ini saya merupakan mahasiswi Politeknik Baja Tegal D3 Prodi Teknik Informatika, saya mengikuti UKM Paduan Suara, salah satu alasan saya masuk UKM paduan Suara yaitu karena saya suka menyanyi.
